# 🛡️ Threat Hunter CLI

A Linux-based Cyber Security and Threat Hunting Toolkit built with Python.

Threat Hunter CLI helps security analysts, SOC analysts, blue teamers, and cybersecurity students perform:

- Threat Hunting
- Incident Response
- IOC Investigation
- Malware Analysis
- Threat Intelligence Lookups
- Security Auditing

## 🚀 Features

### System Monitoring
- System Information
- Open Network Ports
- Running Processes
- Logged In Users
- Failed Login Attempts

### Threat Hunting
- Suspicious Process Scanner
- Rootkit Checker
- File Integrity Monitoring
- Suspicious File Scanner

### Threat Intelligence
- VirusTotal Integration
- AbuseIPDB Integration
- Shodan Integration
- Whois Lookup

### Reporting
- TXT Reports
- JSON Reports
- HTML Reports
